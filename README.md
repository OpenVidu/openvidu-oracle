# openvidu-oracle
Terraform files to deploy all the types of deployments of OpenVidu in Oracle Cloud Infrastructure. You can check the documentation for the deployments in the following links: 
1. [Single Node Community deployment](https://openvidu.io/latest/docs/self-hosting/single-node/oracle/install)
2. [Single Node PRO deployment](https://openvidu.io/latest/docs/self-hosting/single-node-pro/oracle/install)
3. [Elastic deployment](https://openvidu.io/latest/docs/self-hosting/elastic/oracle/install)
4. [High Availability deployment](https://openvidu.io/latest/docs/self-hosting/ha/oracle/install)
